# analise-campanhas-de-marketing
Este é um projeto de banco de dados relacional para análise de campanhas de marketing digital, utilizando dados de uma imobiliária fictícia localizada na cidade de Jaú, SP.

Ferramentas utilizadas
- MySQL Workbench
- Notepad ++ para organização do código.

Estrutura do Projeto:
- Tabela Campanha
- Tabela Leads
- Tabela Interações
- Tabela Conversões.

Como usar:

1. Importe o arquivo através do MYSQL Workbench ou alguma ferramenta SQL.
2. Execute as queries de análise disponíveis:

- Campanhas ativas e orçamento médio.
- Perfil demográfico dos Leads.
- Tempo de interação.
- Tipos de interação por campanha
